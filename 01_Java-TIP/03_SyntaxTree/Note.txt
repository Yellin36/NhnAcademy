 >> 단계별

 1) 정수를 이용한 4칙 연산 계산기
 2) 연산자 순위
      - ( )
      - * /
      - + -
 3) 이진트리를 이용한 구문 트리 생성
      - 연산자 노드 : 이진 연산자가 포함되어 계신이 필요한 노드, 이진 연산자에 필요한 피연산자 노드를 포함함
      - 피연산자 노드 : 연산자 노드의 피연산자가 될 수 있는 노드 / 상수, 변수, 연산자 노드

 1. 이진트리 만들기
  1) 노드 만들기
      > Field
          - value(Integer) : int와 Integer의 차이(primitive인지 reference인지)
          - LeftChildNode
      > Method
          - constructor
          - search(추상) : 탐색을 위함
          - getList(추상) : 모든 자식 노드를 가져오기 위해
  2) 트리구성을 위한 노드 만들기
      > 전위순회
      > 중위순회
      > 후위순회
  3) 각 노드에 add/remove 메소드 추가
      > 왼쪽 자식 노드는 루트노드보다 값이 작습니다.
      > 오른쪽 자식노드는 루트노드보다 값이 큽니다.
      > 정렬된값으로 추가
  4) 제네릭 타입 추가하기
      ~ 자바에서의 제네릭을 추가할때 생기는 오류들이 존재 --> creator 사용
