css의 정의

CSS는 매우 기본적인 텍스트 문서 스타일링에 사용될 수 있습니다.
예를들어, 제목 및 링크의 색상과크기 변경. 또한, 레이아웃을 만드는 데 사용 할 수 있습니다

짧은 인용구 : <q>

CSS의 정의는 사용자에게 문서를 표시하는 방법을 지정하는 언어입니다.
